# wordle-tool

#### Overview
Simple script to help users identify possible word selections for the game Wordle: https://www.powerlanguage.co.uk/wordle/

#### How to Use
- Clone this repository
- Navigate to the repository on command line
- 'python main.py'
- Follow text prompts
